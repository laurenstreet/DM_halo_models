Define the value of \( \alpha \) when using ULDM matched models.

This defines the value of \( \alpha \) in the ULDM matched models. This value of is fixed from the condition \( \rho_{\mbox{sol}}\left(3 \, r_{\mbox{c,sol}}\right) = \rho_{\mbox{Einasto}} \left(3 \, r_{\mbox{c,sol}}\right) \).

Parameters
m22mass of ULDM in units of \( 10^{-22} \, \mbox{eV} \)
msolmass of soliton in units of solar masses
c200Einasto profile variable \( c_{200} \)
V200Einasto profile variable \( V_{200} \)
Returns
float Value of \( \alpha \) to be used for ULDM matched models
