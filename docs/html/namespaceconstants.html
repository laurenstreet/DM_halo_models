def constants.standard ( )

The dictionary containing standard constants.

This defines the dictionary containing standard constants needed during fitting procedure.

Returns
dictionary
Dictionary of standard constants:
  • kmTOGeV in units \([\mbox{km}^{-1} \mbox{GeV}^{-1}] \) = conversion from kilometer to GeV
  • sTOGeV in units \([\mbox{s}^{-1} \, \mbox{GeV}^{-1}]\) = conversion from second to GeV
  • kpcTOGeV in units \([\mbox{kpc}^{-1} \, \mbox{GeV}^{-1}]\) = conversion from kpc to GeV
  • msun in units \([\mbox{GeV}]\) = mass of sun in GeV
  • MP in units \([\mbox{GeV}]\) = Planck mass in GeV
  • rhocrit in units \([\mbox{GeV}^4]\) = critical density of universe in \(\mbox{GeV}^4\)
  • MOND_gdag in units \([\mbox{m} \, \mbox{s}^{-2}]\) = gravitational acceleration constant for gravitational acceleration relation in MOND theory
